About

guided-tour.js lets you create interactive auto-tours for any webpage. Define sections, write step scenarios — the library handles cursor movement, highlights, typing effects and navigation automatically.

  • ✦ Pausable — Space to pause / resume
  • ✦ Keyboard navigation — ← →
  • ✦ Optional terminal overlay
  • ✦ Virtual cursor with hover effects

Features

Virtual Cursor

Smooth animated cursor moves between elements automatically.

Typing Effect

Terminal-style typewriter with optional overlay panel.

Step Scenarios

Declarative steps: hover, highlight, type, scroll, sleep.
