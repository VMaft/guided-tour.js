## Идея и концепция
Что лучше чем просто показ слайдов? Автоматический показ слайдов! А если сделать слайды интерактивными - поулчится сайт. О котором очень удобно рассказать закадрово в субтитрах и **подсветить** самое нужное и значимое.
Проект guided-tour.js - это логический "форк" попытки выйти за границы стандартного показа сайта и простой демонстарции страницы рассказа о себе. Проект предоставляет возможность прокомментировать требуемый всю страницу в привычном, относительно, формате субтитров. Идея авто-туров сама по себе не новая, но главное ведь подача. А в качесве подачи и основной фишки решения выступает легкая интеграция и простая реализация сценариев автотура. 

И так **guided-tour.js** - это легковесная библиотека состоящая из 5 файлов: 
1. guided-tour.js - движок. Здесь описана основаня логика управления демонстрацией, переключению секций автотура, выводу "субтитров", отрисовки элементов автотура на странице во время показа.
2. tour-utils.js - утилиты. Здесь собрана логика анализа и регистрации и проигрывания сценариев, выделдения элементов и обработки остановки тура.
3. tour-cursor.js - библиотека виртуального курсора. Опциональный файл, можно не подключать к основной библиотеке если нет явной необходимости в виртуальном курсоре демонстрации. Так же включает функции выделения элементов, отрисовки и перемещения курсора по странице.  
4. tour-sections-example.js - файл описывающий секции автотура. Опциональный файл описания сценариев для секций необходимый в версии 1.0.0. (В будущем планируется переход на авторегистрацию секций автотура прямо из сценариев).
5. tour-scenario-example.js - файл сценария автодемонстрации. Основной файл с описанием всех действий выполняемых всемы перчисленными выше файлами. Включает комментарий к секции, уникальный идентификатор секции, комментарий к открытию секции. А так же все шаги с действиями которые будут демонстрироваться во время тура. (Предлполагается что сценариев автотура будет несколько под каждую секцию). 

## Зависимости
Все файлы библиотеки должны быть подключены в верном порядке:
```
  <script src="your-sections.js"></script>         // всегда над guided-tour.js для корректного чтения комментариев секций
  <script src="guided-tour.js"></script>           // Основной движок
  <script src="tour-cursor.js"></script>           // Опциональный курсор
  <script src="tour-utils.js"></script>            // Класс утилит
  <script src="yor-tour-scenario-1.js"></script>
                ...
  <script src="yor-tour-scenario-N.js"></script>
```
